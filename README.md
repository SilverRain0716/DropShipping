## ⚠️ 핵심 운영 원칙

1. **크롤러 IP ≠ 판매 계정 IP** — 절대 원칙, 타협 없음
2. **P6(리스크 감시자)를 먼저 열어야 할 상황** = IP 차단 / ToS 경고 / 계정 이상
